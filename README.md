# English-to-NZSL
A HTML, JavaScript based translation tool that moves English to New Zealand Sign Language
This uses JS to translate english text to NZSL drawings.
These drawings were made by the Deaf Studies Research Unit, Victoria University of Wellington
Due to sizing issues you'll have to download the archive from their github repo and extract it.
[NZSL Drawings](https://github.com/ODNZSL/nzsl-dictionary-scripts/releases)
Drag this extracted folder into the project folder and evrything should work

**How to use**
Once downloaded unzip and go through the previously mentioned link to get the images, then unzip the alphabet folder.

Then, view the .html document in a browser and type the english you want to translate in the input box

Click Translate

View the signs

**Limitations**
This code does take into account that some english words such as _is_, or _are_, are only conveyed by context and facial gestures in NZSL. 
However, it does not take into account the syntax and grammar differences. As far as I can tell these would require machine learning and extensive time and money.

Otherwise please report any bugs and I'll try to fix them
